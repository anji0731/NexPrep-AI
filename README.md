# 🚀 NexPrep AI

> **AI-Powered Resume Analyzer & Intelligent Mock Interview Platform**

NexPrep AI is a full-stack AI-powered career preparation platform that helps students and job seekers improve their resumes, evaluate ATS compatibility, generate personalized interview questions, and practice AI-driven mock interviews with instant feedback.

---

## ✨ Features

### 📄 AI Resume Analysis
- Upload Resume (PDF)
- Upload Optional Job Description
- ATS Compatibility Score
- Interview Readiness Score
- Keyword Match Analysis
- Missing Skills Detection
- Resume Strengths & Weaknesses
- Professional Recruiter Feedback
- Career Improvement Suggestions

### 🤖 AI Resume Mock Interview
- Resume-based Interview Questions
- Job Description-based Questions
- AI-generated Technical Questions
- Behavioral Questions
- Real-time AI Evaluation
- Instant Feedback
- Performance Score
- Final Interview Report

### 💼 Technical Practice
- AI-generated Technical Questions
- Coding Practice
- Skill-based Preparation

### 👨‍💼 HR Practice
- HR Interview Questions
- Behavioral Assessment
- Communication Evaluation

### 📊 Dashboard
- Resume History
- Interview History
- Progress Tracking
- Performance Analytics

# 🚀 Quick Start

Clone Repository

```bash
git clone https://github.com/anji0731/NexPrep-AI.git
```

Go to project

```bash
cd NexPrep-AI
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Runs at

```
http://localhost:5173
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Runs at

```
http://localhost:8000
```

Swagger

```
http://localhost:8000/docs
```
